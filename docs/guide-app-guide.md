# ガイド スタッフアプリ 使い方ガイド

## はじめに

このアプリは、**販売スタッフがお客様の接客をサポートするためのツール**です。
スマートフォンやタブレットから使用でき、商品情報の確認・おすすめ提案・比較がすぐにできます。

---

## 画面の流れ

```
トップ画面
  ├─ カテゴリを選ぶ
  │    ├─ 商品一覧
  │    │    └─ 商品詳細
  │    └─ ウィザード（おすすめ提案）
  │         └─ おすすめ結果 → 比較
  └─ 検索（型番・製品名・メーカー）
```

---

## 1. トップ画面

アプリを開くと最初に表示される画面です。

### カテゴリ一覧

登録されている商品カテゴリ（電子レンジ・冷蔵庫など）がタイル形式で並んでいます。
**見たいカテゴリをタップ**すると、そのカテゴリの商品一覧に進みます。

### 商品を検索する

画面上部の検索欄に**製品名・型番・メーカー名**を入力すると、リアルタイムで検索結果が表示されます。

- 2文字以上入力すると自動的に検索が始まります
- 結果の商品をタップすると商品詳細に移動します
- 右端の `✕` を押すと検索をリセットします

### お気に入り

過去にお気に入り登録した商品が画面上部にまとめて表示されます。
▲ / ▼ ボタンで折りたたみ・展開ができます。

---

## 2. カテゴリ画面（商品一覧）

カテゴリをタップすると、そのカテゴリの商品が一覧で表示されます。

### 商品カードを見る

各カードには以下の情報が表示されています：

| 表示内容 | 説明 |
|---|---|
| 型番（大きい文字） | 商品の型番 |
| 製品名・メーカー | 小さい文字で表示 |
| 価格 | 右端に表示 |

**カードをタップ**すると商品詳細画面に進みます。

### 商品を絞り込む

一覧上部の入力欄に製品名・型番・メーカーを入力すると、リスト内を絞り込めます。

### 商品を比較する

各カード右上の **`+` ボタン**をタップすると比較リストに追加されます。

- 最大 **5件** まで選択できます
- 選択した件数は画面下部のバーに表示されます
- **「比較する →」ボタン**をタップすると比較画面に進みます
- 「クリア」ボタンで選択をリセットできます

> **ポイント：** 選択した商品はページを移動しても保持されます。

---

## 3. ウィザード（お客様おすすめ提案）

カテゴリ画面の上部にある **「お客様と一緒に商品を探す」** ボタンをタップするとウィザードが起動します。

### ウィザードの使い方

1. **お客様と一緒に画面を見ながら質問に答えていきます**
   例：「お手入れのしやすさは重視しますか？」

2. **選択肢をタップ**して次の質問へ進みます
   絶対条件（必須）にしたい項目は **「絶対条件にする」** をオンにします

3. 全質問が終わると **「お客様におすすめ」** の商品が表示されます

### おすすめ結果の見方

- **マッチしたキーワード**（青いタグ）：お客様の条件と一致した機能
- **なぜおすすめ？**：おすすめの理由
- 商品名をタップすると詳細ページに移動します

### おすすめ商品を比較する

おすすめ商品が2件以上ある場合、**「比較する」ボタン**が表示されます。
タップするとおすすめ商品を横並びで比較できます。

### 選択をリセットする

**「最初からやり直す」** をタップすると、ウィザードをリセットして最初から始められます。

---

## 4. 商品詳細画面

商品カードをタップすると詳細画面が表示されます。

### 画面上部（ヘッダー）

| ボタン | 説明 |
|---|---|
| `← 戻る` | 前の画面に戻る |
| 価格（タップ可） | タップすると価格を変更できます（当日限りの一時的な変更） |
| `♡ / ♥` | お気に入りへの追加・削除 |
| `解説` | 製品解説ポイント（ホットスポット）を表示 |
| `QR` | お客様にシェアするQRコードを表示 |

> **価格の変更について：** 価格をタップして変更すると、その端末のセッション中のみ反映されます。アプリを閉じると元の価格に戻ります。

### 商品画像とホットスポット

商品画像の上に**番号付きのドット**が表示されている場合、タップすると機能の説明が出てきます。
下部のタブをタップして切り替えることもできます。

### 製品解説ポイント（解説モーダル）

ヘッダーの **「解説」ボタン**をタップするとフルスクリーンで製品解説ポイントを確認できます。

- 画像上のドットをタップすると説明とフレーズが表示されます
- 画面下部のタブからも切り替えられます
- お客様に見せながら説明するのに最適です

### おすすめ接客トーク

青いカードに**接客用のトーク例**が書かれています。
そのまま読み上げるか、参考にしてお客様に説明しましょう。

### 売りポイント

この商品の**特徴や推しポイント**がまとめられています。

### メリットの伝え方

専門用語とその**わかりやすい説明**が一覧になっています。
右の **`i` ボタン**をタップするとドロワーで詳細が確認できます。

### 製品スペック

各スペック項目の右にある **`i` ボタン**をタップすると、その値の意味をわかりやすく説明したドロワーが開きます。

### 他製品との比較

登録済みの比較データがある場合、このセクションが表示されます。
各カードには「旧モデル」または「競合」バッジと、比較対象の型番が表示されます。

**カードをタップ**するとフルスクリーンで詳細が開きます：
- **接客ポイント**：お客様への説明で使えるポイントをまとめた文章
- **比較表**：項目ごとの本製品と比較対象の値（本製品列は青く強調）

> **注意：** 比較データはブログ記事をAIが自動分析したものです。重要なスペックはメーカー公式ページでご確認ください。

### QRコード

ヘッダーの **「QR」ボタン**をタップすると、メーカー公式ページのQRコードが表示されます。
お客様自身のスマートフォンで読み取ってもらうと、詳細情報をご自身で確認いただけます。

---

## 5. 比較画面

複数の商品を並べてスペックを比較できます。

### 比較画面の使い方

1. カテゴリ画面またはウィザード結果からスペックを確認したい商品を選択
2. **「比較する →」ボタン**をタップ
3. 商品がヘッダーに横並びで表示され、スペック項目を縦に比較できます

### 比較画面で確認できる情報

- 商品画像・型番・価格
- 各スペック項目の値（差がある行は黄色ハイライト）
- 生活でのメリット（glossary）
- 売りポイント・接客トーク

商品名をタップすると、その商品の詳細ページに移動します。

### 他製品との比較ポイント

「接客トーク・売りポイント」の各製品カード下部に、比較中の他の商品との比較ポイントが表示される場合があります。

- 比較対象の型番と「旧モデル」「競合」バッジが表示されます
- タップするとフルスクリーンで接客ポイント（summary）と比較表が確認できます
- データは商品登録時にAIが自動抽出したものです

> **注意：** 比較データはブログ記事をもとにAIが生成したものです。正確な数値はメーカー公式ページでご確認ください。

---

## 6. お気に入り機能

### お気に入りに追加する

商品詳細画面のヘッダーにある **`♡` ボタン**をタップするとお気に入りに登録されます（`♥` に変わります）。

### お気に入りを確認する

- **トップ画面**：全カテゴリのお気に入りが表示されます
- **カテゴリ画面**：そのカテゴリのお気に入りが一覧上部に表示されます

> **注意：** お気に入りはブラウザのローカルストレージに保存されます。ブラウザのデータをクリアすると消えますのでご注意ください。

## よくある操作パターン

### パターン A：型番がわかっているとき

1. トップ画面の検索欄に型番を入力
2. 検索結果から商品をタップ
3. 接客トーク・スペックを確認

### パターン B：お客様の用途に合う商品を探したいとき

1. カテゴリを選ぶ
2. **「お客様と一緒に商品を探す」** をタップ
3. お客様と一緒にウィザードを進める
4. おすすめ商品を確認・比較

### パターン C：2〜3機種で迷っているとき

1. カテゴリ画面で各商品カードの **`+`** をタップして選択
2. **「比較する →」** をタップ
3. スペックを横並びで確認

### パターン D：後でもう一度見たいとき

1. 商品詳細の **`♡`** をタップしてお気に入りに登録
2. トップ画面に戻ると「お気に入り」欄に表示される

---

## 困ったときは

| 症状 | 対処 |
|---|---|
| 商品が見つからない | 型番の表記を確認（ハイフンなしで入力） |
| ウィザードでおすすめが出ない | 絶対条件を減らして再度試す |
| 価格が違う | 価格をタップして当日の価格に変更する |
| 画面が固まった | ブラウザを更新（リロード）する |
