# 商品紹介アプリ (guide-app) 開発ログ

## アプリ概要

家電量販店の新人スタッフが**タブレットを使って接客しながら商品を一緒に選べる**商品紹介ツール。
Supabase DBに登録された商品の接客トーク・スペック・用語解説を表示する。

- **フォルダ**: `guide-app/`
- **技術スタック**: Next.js / TypeScript / Tailwind CSS v4 / Supabase
- **起動**: `cd guide-app && npm run dev` → http://localhost:3000

---

## クイックスタート

### 必要な環境

- Node.js 18 以上
- npm

### セットアップ

```bash
cd guide-app
npm install
```

`.env.local` をプロジェクトルートに作成して以下を記入：

```
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```

### 起動

```bash
npm run dev
```

→ http://localhost:3000 で起動します。

### iPad へのインストール（PWA）

1. Safari で本番 URL（https://guide-app-beta.vercel.app/）を開く
2. 画面下の共有ボタン →「ホーム画面に追加」
3. アイコンをタップするとフルスクリーンで起動（アドレスバーなし）

## 機能の使い方

### カテゴリ選択ホーム（`/`）

起動するとカテゴリのグリッドが表示されます。
カテゴリをタップすると、そのカテゴリの商品一覧へ移動します。

ページ上部の検索バーでは全カテゴリをまたいで商品を検索できます（製品名・型番・メーカーで絞り込み）。

### 商品一覧（`/categories/[id]`）

| 操作 | 方法 |
|------|------|
| 絞り込み | 上部の検索バーに入力（製品名・型番・メーカー） |
| 比較に追加 | カード右上の「＋」ボタンをタップ（最大5件） |
| 比較を開始 | 下部に出る「X件を比較する」バーをタップ |
| 価格を編集 | 価格の数字をタップ → 直接入力（セッション中のみ有効） |
| ウィザードを使う | 「お客様と一緒に商品を探す」ボタン → キーワードを選んでおすすめ表示 |

### 商品詳細（`/products/[id]`）

| 要素 | 内容 |
|------|------|
| ご案内例（接客トーク） | お客様への説明文 |
| 売りポイント | 商品の主な特長 |
| メリットの伝え方 | 各機能が生活でどう役立つかの解説 |
| 製品スペック | 詳細なスペック一覧 |
| 他製品との比較 | 登録済みの比較データをカード形式で表示。タップするとフルスクリーンモーダルで接客ポイント→比較表を確認できる |
| 解説ボタン | 商品画像上のホットスポット解説を表示（登録済み商品のみ表示） |
| QRコードを表示 | お客様のスマホへ商品ページURLをQRコードで共有（`source_url` 登録済みの商品のみ） |

### ウィザード（商品一覧内）

「お客様と一緒に商品を探す」ボタンをタップして起動。
キーワードタイルをタップして選択すると、条件に合ったおすすめ商品がリアルタイムで右に表示されます。
おすすめ商品が2件以上の場合は「比較する」ボタンも表示されます。

### 比較ページ（`/compare`）

選択した商品のスペック・価格・売りポイントを横並びで比較できます。
差があるスペック行は黄色でハイライト表示され、最安商品には「最安」バッジが付きます。
価格はここでもタップ編集が可能です（セッション中のみ有効）。

「接客トーク・売りポイント」セクションの各製品カードの下部に**「他製品との比較ポイント」**が表示される場合があります。これは `product_comparisons` に登録された比較データから、比較中の商品ペアに該当するものを自動で引き当てたものです。タップするとフルスクリーンで接客ポイント（summary）と比較表が表示されます。

> **注意：** 比較ポイントはブログ記事をもとにAIが自動生成したデータです。重要なスペックはメーカー公式ページでご確認ください。
