# ガイド 管理者アプリ (guide-admin) 使い方ガイド

## はじめに

このアプリは、**スタッフアプリ（guide-app）に表示される商品情報を登録・管理するための管理者専用ツール**です。
AIが公式サイトから自動でスペックや接客トークを抽出するため、手入力の手間を大幅に削減できます。

> **アクセス**: 管理者用URLをブラウザで開き、発行済みのメールアドレス・パスワードでログインしてください。

---

## 画面の流れ（全体像）

```
ログイン
  └─ ダッシュボード（登録商品の一覧・確認）
       ├─ 商品登録       ← 新しい商品を追加するメインフロー
       ├─ 商品一覧       ← 登録済み商品の管理・編集
       ├─ カテゴリ管理   ← カテゴリとAI設定を管理
       ├─ ダッシュボード ← 登録状況の確認
       ├─ ユーザー管理   ← アカウント管理（スーパー管理者のみ）
       └─ 設定           ← ブロックドメインの管理
```

---

## 1. ログイン

1. 管理者URLをブラウザで開く
2. メールアドレスとパスワードを入力して「ログイン」をクリック
3. ログイン後はダッシュボードが表示されます

> ログアウトはナビバー右端の「ログアウト」ボタンで行えます。

---

## 2. 商品登録（メインフロー）

新しい商品をスタッフアプリに登録する手順です。ナビから「**商品登録**」を選びます。

### ステップ 1：モードを選ぶ

画面右上のボタンで登録方法を選択します。

| モード | 使う場面 |
|---|---|
| **新規登録**（デフォルト） | 初めて登録する商品。または登録済み商品を再解析する場合も同じモードを使用 |
| **バッチ登録** | 複数商品を一度に登録したいとき |
| **手動登録** | AI解析なしで直接フォームに入力して登録したいとき |

> **登録済み商品を再解析したい場合：** 新規登録モードで同じ型番を入力すると「この型番はすでに登録されています」という警告モーダルが表示されます。「上書きして再解析」ボタンを押すと、既存データを上書きして解析を続行できます。

### ステップ 2：カテゴリを選ぶ

登録する商品のカテゴリ（例：電子レンジ・オーブンレンジ）を選択します。前回使ったカテゴリが自動で選択されます。

### ステップ 3：型番を入力してURL検索

商品の**型番**（例：`ER-D7000B`）を入力して「**URL候補を探す**」ボタンを押します。

URLが **3つのセクション** に分類されて表示されます：

| セクション | 内容 | デフォルト |
|---|---|---|
| **公式サイト** | メーカー公式ページ | チェックあり |
| **比較情報** | 比較ブログ記事（monomaniaなど） | チェックあり |
| **その他** | YouTube・価格比較サイトなど | チェックなし |

不要なURLのチェックを外してから「**解析する**」ボタンを押します。

> **型番のハイフンについて**: ハイフンがあってもなくても認識されます。保存時はハイフンなしで統一されます。

### ステップ 4：解析する

「**解析する**」ボタンを押すと、AIが選択したURLから以下の情報を自動抽出します：

- 商品名・メーカー
- 接客トーク（スクリプト）
- 売りポイント（unique selling point）
- スペック一覧
- 用語解説（glossary）
- ウィザード用スペック（カテゴリ対応済みの場合はルールベースで高精度抽出）
- 比較情報（比較ブログURLが含まれる場合）→ 旧モデル比較・競合比較データを自動抽出

> 解析には 10〜60 秒かかる場合があります。エラーが出た場合は再度試してください（Gemini の一時的な混雑の場合があります）。

### ステップ 5：内容を確認して保存

解析結果がプレビューされます。

- **価格欄**：「タップして金額を入力」をクリックして当日の価格を手入力してください（家電の価格は頻繁に変わるため手動管理）
- **比較情報**：比較ブログが取得できた場合、旧モデル比較・競合比較の表が表示されます。内容はブログ記事をもとに自動生成されたものです

内容を確認して「**保存する**」をクリックします。

### ステップ 6：ウィザードスコアを設定

保存後、商品のウィザードスコア設定画面になります。
各キーワード（例：「温め・解凍」「お手入れ」）に対してスコアを設定します。

| スコア | 意味 |
|---|---|
| 1 | 非搭載（ウィザードのハードフィルターで除外される） |
| 2 | エントリー（一応ついている程度） |
| 3 | 搭載（標準的に搭載） |
| 4 | 高精度（高精度に搭載） |
| 5 | ★推し（この商品が特に推している機能 — 1商品につき1つだけ） |

「**自動生成**」ボタンでスペックから自動判定することもできます。
各スコアには「なぜ？」の理由を接客トーンで入力してください。

## 3. 商品一覧

ナビから「**商品一覧**」を選ぶと、登録済み全商品が一覧で表示されます。

### バッジの見方

各商品名の横に表示されるバッジの意味：

| バッジ | 意味 |
|---|---|
| 🟢 スコアあり | ウィザードスコアが全キーワード入力済み |
| 🟠 未入力スコアあり | ウィザードで新しいキーワードが追加されたが未入力 |
| ⚪ スコアなし | ウィザードスコアが未登録 |
| 🟡 要更新 X項目 | カテゴリのスペック項目のうち未入力があるもの |

> **「未入力スコアあり」が表示されている商品**: ウィザードのキーワードが追加されたため、スコアの入力が必要です。商品を開いてウィザードスコアセクションから入力してください。

### 商品を絞り込む

- **カテゴリで絞り込み**: 上部のドロップダウンでカテゴリを選択
- **テキスト検索**: 型番・製品名・メーカーで絞り込み

### 商品を編集する

商品名をクリックすると編集ページに移動します。

### 一括削除

1. 削除したい商品の左端のチェックボックスをオンにする
2. ヘッダーのチェックボックスで全選択も可能
3. 「選択したX件を削除」ボタンをクリック

> **注意**: 削除は元に戻せません。

---

## 4. 商品編集ページ

商品一覧から商品をクリックすると編集ページが開きます。

### 編集できる項目

| セクション | 内容 |
|---|---|
| **基本情報** | カテゴリ・型番・メーカー・製品名・価格・商品画像 |
| **接客トーク** | スタッフアプリに表示される接客用のトーク文 |
| **売りポイント** | この商品の特徴・強み |
| **スペック** | カテゴリで定義したスペック項目の値 |
| **メリットの伝え方** | 用語と分かりやすい説明のペア |
| **ウィザードスコア** | キーワードごとのスコア（1〜5）と理由 |
| **ホットスポット** | 商品画像上の解説ポイントの登録・編集 |

### スペックセクション

- 🟡 アンバー背景の項目は「カテゴリで定義されているが未入力」のもの
- 「手入力」バッジがついた項目は手動で値を変えたもの

### ウィザードスコアセクション

- 🔴 赤背景 + 「現在この項目は削除されています。」→ ウィザードから削除されたキーワード（不要なら「✕」で削除可）
- 🟡 アンバー背景 → 未入力のキーワード（理由を入力してください）
- **自動生成**: スペックデータからスコアを自動計算
- **スコアを保存**: 入力したスコアを保存（スコア5が2つ以上あると保存できません）

### ホットスポットセクション

商品画像上の特定の部位に解説を設定できます。

**編集モード**:
1. 「**編集モード**」が選択された状態で画像をクリック
2. クリックした位置にホットスポットが仮配置される
3. ラベル・説明・接客フレーズを入力して「保存」

**スタッフ画面プレビュー**:
- 「**スタッフ画面プレビュー**」ボタンでスタッフアプリでの表示を確認できます
- ドットをタップして説明が正しく表示されるか確認してください

> **ポイント**: 「スタッフ画面プレビュー」と実際のアプリ表示は同じ座標系で一致しています。

---

## 5. カテゴリ管理

ナビから「**カテゴリ管理**」を選びます。

### カテゴリとは

商品を分類するグループです（例：電子レンジ・オーブンレンジ、冷蔵庫など）。
カテゴリごとに **スペック項目** と **AIへの接客トーク方針** を設定できます。

### カテゴリを追加する

1. 右上の「**+ 新規追加**」をクリック
2. 以下を入力して「追加する」をクリック

| 項目 | 説明 | 例 |
|---|---|---|
| **カテゴリ名** | 商品の種類名 | `電子レンジ・オーブンレンジ` |
| **スペック項目** | AIが抽出するスペックの項目名をカンマ区切りで入力 | `定格高周波出力, 庫内容量, 最高温度` |
| **接客トーク方針** | AIへのヒント（どんな観点でトークを作るか） | `調理の手軽さ・時短・プロ級仕上がりを強調すること` |

### ウィザードを設定する

カテゴリ一覧の各カテゴリに「**ウィザード**」ボタンがあります。
クリックするとそのカテゴリのヒアリングウィザードの編集ページへ移動します。

---

## 6. ウィザード編集ページ

カテゴリ管理から「ウィザード」ボタンで移動します。

### ウィザードとは

スタッフアプリでお客様と一緒に使う「おすすめ商品を絞り込む質問フロー」です。
ステップ（質問）と選択肢（キーワード）を設定します。

### スコアの意味（凡例）

ページ上部にスコア1〜5の意味が表示されています。
スコアはウィザードの選択肢（キーワード）と商品のマッチングに使われます。

| スコア | 意味 |
|---|---|
| 1 | 非搭載 → ハードフィルターで除外 |
| 2 | エントリー |
| 3 | 搭載 |
| 4 | 高精度 |
| 5 | ★推し（商品登録ページで手動設定） |

### ステップを追加する

1. 「**+ ステップを追加**」をクリック
2. **質問文** を入力（例：「お手入れのしやすさは重視しますか？」）
3. **選択肢（キーワード）** を入力（例：「重視する」「どちらでも」）
4. 補足テキストを入力（任意）

### 保存時の自動同期

ウィザードを保存すると、**そのカテゴリの全商品に新しいキーワードが自動で追加**されます。
商品一覧で「未入力スコアあり」バッジが表示されたら、各商品のスコアを入力してください。

> **キーワードを削除した場合**: 既存の商品スコアは残りますが、商品編集ページで「現在この項目は削除されています」と赤く表示されます。不要であれば商品編集ページから削除してください。

---

## 7. ダッシュボード

ナビから「**ダッシュボード**」を選びます。

登録されている商品を確認できます。

### フィルター

| フィルター | 表示内容 |
|---|---|
| すべて | 全商品 |
| 直近7日 | 過去7日以内に登録した商品 |
| 直近30日 | 過去30日以内に登録した商品 |
| 画像なし | 商品画像が未設定の商品 |

---

## 8. 設定（ブロックドメイン）

ナビから「**設定**」を選びます。

### ブロックドメインとは

AI検索（URL候補の自動検索）から除外したいドメインを登録できます。
例えばレビューサイトや価格比較サイトを登録しておくと、公式サイト以外のURLが候補に上がりにくくなります。

### ドメインを追加する

1. 「ドメインまたはURL」欄に追加したいドメインを入力（URLを貼り付けても自動でドメインに変換されます）
2. 「理由」を入力（任意）
3. 「**追加**」ボタンをクリック

### ドメインを削除する

一覧の「削除」ボタンをクリックすると確認後に削除されます。

---

## 9. ユーザー管理（スーパー管理者のみ）

スーパー管理者アカウントでログインした場合のみナビに表示されます。
管理者アカウントの追加・削除・権限変更を行えます。

---

## よくある操作パターン

### パターン A：新商品を1件登録する

1. ナビ「商品登録」→ 新規登録モードを確認
2. カテゴリを選択
3. 型番を入力 →「AI検索」→ URLを選択 →「解析する」
4. プレビューを確認 →「保存する」
5. ウィザードスコアを設定 →「スコアを保存」
6. 商品編集ページでホットスポットを設定（任意）

### パターン B：既存商品の情報を更新する

1. ナビ「商品登録」→ 新規登録モードのまま型番を入力
2. 「URL候補を探す」→ 登録済み商品の警告モーダルが表示される
3. 「上書きして再解析」をクリック →「解析する」
4. 内容を確認して「保存する」

### パターン C：ウィザードに新しい質問を追加する

1. ナビ「カテゴリ管理」→ 対象カテゴリの「ウィザード」をクリック
2. 「+ ステップを追加」→ 質問と選択肢を入力
3. 「保存する」（自動で全商品にキーワードが追加される）
4. ナビ「商品一覧」→「未入力スコアあり」の商品を開いてスコアを入力

### パターン D：特定の商品を素早く見つけて編集する

1. ナビ「商品一覧」→ カテゴリで絞り込み or テキスト検索
2. 商品名をクリックして編集ページへ

---

## 困ったときは

| 症状 | 対処 |
|---|---|
| AI検索でURLが見つからない | 型番を変えて試す。またはURLを手動で貼り付ける |
| 解析エラーが出る（503） | Geminiの一時的な混雑のため、しばらく待って再試行 |
| 再解析ボタンが見当たらない | 再解析ボタンは廃止されました。新規登録モードで同じ型番を入力すると警告モーダルが出ます |
| ブロックドメインの追加に失敗する | Supabaseのテーブルが作成されているか確認。管理者に連絡 |
| スコア5が保存できない | スコア5は1商品につき1つのみ。2つ以上設定されていないか確認 |
| 「現在この項目は削除されています」が表示される | ウィザードから削除されたキーワード。不要なら商品編集画面の「✕」で削除 |
| 商品画像が表示されない | 画像URLが有効か確認。または商品編集ページから再設定 |
